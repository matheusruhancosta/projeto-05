Relatorio de Estagio

O objetivo geral do estágio foi aplicar os conhecimentos teóricos adquiridos ao longo do curso de Informática, focando em atividades práticas relacionadas à programação, manutenção de sistemas e gestão de infraestrutura de TI, visando o desenvolvimento profissional na área.

Desenvolve habilidades praticas na area de desenvolvimento

1.1.2. Objetivos Especificos

2. Atividades Realizadas

Nesta seção, são descritas as atividades realizadas durante o estágio, detalhando o que foi feito, as ferramentas utilizadas e os resultados alcançados.

2.1. Atividade 01 – Desenvolvimento de Scripts para Automação de Processos

A primeira atividade envolveu o desenvolvimento de scripts em Python para automação de processos repetitivos, como backups diários de dados e a limpeza de logs do sistema.

2.2. Atividade 02 – Otimização de Banco de Dados

A segunda atividade consistiu na otimização de consultas SQL e a indexação de tabelas no banco de dados MySQL, melhorando o desempenho das consultas.

O estágio proporcionou uma vivência prática importante, permitindo o desenvolvimento de habilidades técnicas e a melhoria de processos na empresa. A experiência fortaleceu minha formação profissional e me preparou para futuros desafios na área de TI.
